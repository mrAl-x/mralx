Security & privacy by design

Secure, scalable, cloud native

As we run completely without passwords, there are none stored in our databases, making credential breaches a thing of the past. Each Hanko project consists of a bunch of containers that run in an isolated namespace with its own databases. If required, we can also provide private cloud environments.
